# Sales Performance BI Dashboard

## Overview
This project analyzes sales performance, profitability, and trends using a Power BI dashboard.

The goal is to support business decision-making through KPI tracking and data visualization.

## Business Problem
The business needed visibility into sales performance, profit margins, and regional trends to support strategic decisions.

## Solution
- Cleaned and transformed data using Power Query
- Built a data model with fact and dimension tables
- Created DAX measures for KPIs
- Designed interactive dashboards in Power BI

## Key Insights
- Identified top-performing regions and products
- Analyzed profit margins and discount impact
- Highlighted sales trends over time

## Tools Used
- Power BI
- DAX
- Power Query

## Business Analysis Concepts
- KPI Definition
- Data Modeling
- Trend Analysis
- Decision Support
